## Mapas Brasil-Escócia para Colaborações Acadêmicas (Mapas BECA)
[Voltar para a página principal](../index.md)

No mapa interativo abaixo, você pode (_alternativa para dispositivos móveis_ entre parênteses):
- _rolar_ (_pinçar_) para aumentar ou reduzir o zoom em regiões;
- _clicar e arrastar_ (_deslizar_) para se mover entre regiões;
- _clicar_ (_tocar_) em um cluster de marcadores para expandí-lo;
- _posicionar o mouse sobre_ (_tocar em_) um marcador para visualizar sua informação associada.
